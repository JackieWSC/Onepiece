Git Command

Common

Show the current status

git status

Stage the files

git add version4.txt

git add .

Commit all changes and Add comment

git commit -a ""

git commit -m "comment"

Discard change in working directory

git checkout -- aaaa.txt

Remove the files

git rm --cached

Push to remote

git push

Diff the local change

git diff --cached

Branch

Check branch status

git branch

Check all local/remote branch status

git branch -r

Create new branch "temp"

git branch temp

Remove branch "temp"

git branch -D temp

Switch out branch

git checkout temp

Pull the code from remote branch

git pull origin master

Add new branch

git checkout -b feature_branch_name

... edit files, add and commit ...

git push -u origin feature_branch_name

Remove branch

git branch -D bugfix

git push origin --delete branchName

Mark Tag

Create tag

git tag -a Build002 -m 'Build002'

git tag -a Build001 7c5f24d

git push origin Build002

git push origin --tags

Remove tag

git tag -d tagName

git push origin :refs/tags/tagName

Rebase

rebase i example
https://www.atlassian.com/git/tutorials/rewriting-history/git-rebase-i
If the branch temp is different with master now

git checkout temp

git rebase master

Update private feature branch to latest

git pull --rebase origin master

If there is conflict, edit those conflict files and

git rebase --continue

Log

show log/show top 5 log

git log

git log -5

show log detail

git log

git log --stat -5

show hash and commit msg only

git log --pretty=oneline -5

show graph

git log --graph -5

list the commited files in ac55.... commit

git show --pretty="format:" --name-only ac55

Config

show git config

git config --list

Other

cherry-pick

git cherry-pick 7300a6

show clean up files listed

git clean -n -d -x

git clean -n -d

review the differences with head

git log HEAD..origin/master

unstage the files

git reset HEAD...

Stashing

git stash

git stash list

git stash apply

Example

Your branch and 'origin/master' have diverged, switch to new master

git branch temp

git fetch origin

git reset --hard origin/master

git reset --hard HEAD~3

branch practice

git checkout master

git branch temp

git checkout temp

git add .

git commit -m "Jackie test"

You need to create a local branch that tracks a remote branch. The following command will create a local branch named daves_branch, tracking the remote branch origin/daves_branch.

git branch -r

git checkout --track origin/daves_branch
